# CAPACIDADES DEL ASISTENTE IA - ACUAMED

## DATOS DISPONIBLES

### Tablas con datos reales:
1. **consumptions** (42,221 registros) - Datos de consumo de agua
2. **locations** (9 registros) - Sistemas de desalación
3. **file_uploads** (1 registro) - Historial de cargas

### Tablas vacías (sin datos):
- sensors (0 registros)
- water_records (0 registros)
- anomaly_records (0 registros)

### Ubicaciones disponibles:
- Sistema Dalías (D)
- Sistema Valdelentisco (D+D)
- Sistema Torrevieja (D+D)
- Sistema Águilas (D+D)
- Sistema Mutxamel (D+D)
- Sistema Oropesa (D+D)
- Sistema Moncófar (D+D)
- Sistema Sagunto (D+D)
- Sistema Carboneras (D+D)

### Campos disponibles en consumptions:
- location_id (relación con ubicaciones)
- period_start / period_end (fechas)
- volume_m3 (volumen en metros cúbicos)
- consumption_type (tipo de explotación/abastecimiento/distribución)
- subscriber_code (código de subscriber)
- is_estimated (si es estimado)
- raw_data (datos originales del Excel en JSON)

## LIMITACIONES TÉCNICAS

### La IA solo genera consultas SQL SELECT
- NO puede INSERT, UPDATE, DELETE
- NO puede modificar datos
- NO ejecuta comandos del sistema

### Depende de los datos cargados
- Si no hay datos de un período, no puede responder
- Si una tabla está vacía, no puede consultarla
- Los resultados dependen de la calidad de la carga

### Formato de respuestas
- Genera explicaciones en español
- Proporciona los datos en tabla
- Incluye la SQL generada (para transparencia)
